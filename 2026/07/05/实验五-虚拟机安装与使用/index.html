实验五 虚拟机安装与使用

Posted on 2026-07-05 Edited on 2026-07-12 In 实验报告

一、实验目的

  1. 使用虚拟机工具（VirtualBox）安装Linux系统（Ubuntu Server）；
  2. 通过克隆技术构建一个多机虚拟化环境；
  3. 了解和掌握基于虚拟机的开发环境的基本构建方法和配置。

二、实验环境

  • 操作系统：Windows 10/11
  • 硬件要求：CPU支持VT-x/AMD-V；内存≥8GB
  • 所需软件：Oracle VirtualBox、Ubuntu Server 22.04 LTS ISO镜像

三、实验过程

步骤一：下载并安装VirtualBox

步骤二：创建模板虚拟机

内存3096MB，VDI动态分配25GB，双网卡（NAT + Host-Only）。

步骤三：Ubuntu系统安装与静态IP配置

步骤四：克隆虚拟机

步骤五：连通测试

四、实验结果

成功搭建多机Linux集群，所有虚拟机互通正常。

五、知识总结

网络模式 访问权限 适用场景
NAT 虚拟机可上外网 日常下载更新
桥接模式 与宿主机同局域网 外部直连虚拟机
Host-Only 仅宿主机互通 隔离内网集群

六、心得体会

掌握虚拟化软件安装、Linux虚拟机部署、多机克隆与静态网络配置。
