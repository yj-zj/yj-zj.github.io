实验三 代码编辑器

Posted on 2026-07-03 Edited on 2026-07-12 In 实验报告

一、实验目的

  1. 安装Visual Studio Code（VS Code）文本代码编辑器；
  2. 安装和配置Markdown相关插件，掌握代码编辑器插件的基本概念；
  3. 学习Markdown基础语法，使用VS Code完成Markdown文档的编辑和渲染任务。

二、实验环境

  • 操作系统：Windows 10/11
  • 网络：需要互联网连接以下载软件和插件
  • 所需软件：Visual Studio Code、VS Code插件（Markdown All in One、Markdown Preview Enhanced等）

三、实验过程

步骤一：下载并安装VS Code

  1. 打开浏览器访问VS Code官网，点击Windows安装包下载。
  2. 双击安装程序启动向导，在附加任务页面全部勾选实用选项。
  3. 一路点击下一步完成安装，双击桌面快捷方式启动编辑器。

步骤二：熟悉VS Code整体界面

启动后依次认识五大核心区域：

  1. 左侧活动栏：资源管理器、搜索、扩展、Git等功能入口；
  2. 侧边栏：对应功能详细面板；
  3. 中央编辑区：代码/文档编辑窗口；
  4. 底部面板：终端、输出、报错信息；
  5. 底部状态栏：文件编码、Git分支、换行格式。

步骤三：安装Markdown核心插件

  1. 快捷键 Ctrl+Shift+X 打开扩展市场；
  2. 搜索 Markdown All in One，点击安装；
  3. 搜索 Markdown Preview Enhanced 完成安装。

步骤四：编辑器个性化基础设置

  1. 快捷键 Ctrl+, 打开设置面板；
  2. 搜索 iles.autoSave，设置为afterDelay开启自动保存；
  3. 搜索 editor.wordWrap，设置为on实现文字自动折行。

步骤五：新建并编写Markdown测试文档

  1. 快捷键 Ctrl+N 新建空白文件，按 Ctrl+S 保存为 est.md；
  2. 在文档中输入标准Markdown演示内容。

  1. 使用插件提供预览快捷键，分屏实时查看渲染后的页面效果。

四、实验结果

成果说明

  1. 成功完成VS Code完整安装并配置系统PATH，命令行可直接调用code指令；
  2. 成功安装两款主流Markdown增强插件，全部功能正常可用；
  3. 完成编辑器自动保存、自动折行等优化配置，提升写作体验；
  4. 独立编写包含标题、加粗、列表、代码块的标准Markdown文档，预览渲染无格式错乱。

五、知识总结

代码编辑器与IDE对比

类型 核心特点 代表工具
代码编辑器 轻量启动、按需插件扩展，仅专注文本编辑 VS Code、Notepad++、Sublime
IDE集成开发环境 内置编译、调试、项目管理全套工具，体积更大 PyCharm、IDEA、Eclipse

Markdown基础语法速记

语法标记 效果
# 一级标题 页面大标题
## 二级标题 章节标题
文字 加粗文本
- 内容 无序列表
`语言代码 代码块高亮

本次实验常用快捷键

快捷键 功能
Ctrl+Shift+X 打开扩展市场
Ctrl+, 打开设置面板
Ctrl+N 新建文件
Ctrl+S 保存文件
Ctrl+K V Markdown分屏预览

六、心得体会

技术层面

熟悉了VS Code完整使用流程，掌握Markdown一些标准语法与预览、导出操作。

思政层面

VS Code作为微软开源免费工具，十分便利。国内也有国产化替代工具与本土插件社区正在发展。作为计算机专业学生，既要熟练掌握主流开发工具，也要主动关注国产编程软件发展。
